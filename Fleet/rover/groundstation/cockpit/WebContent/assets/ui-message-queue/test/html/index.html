UI-Message-Queue Demo

This page is a demo of the UI Message Queue - a Javascript implementation of a message queue allowing you to push messages (String) to the queue and display in via a FIFO (First In First Out) method. @See https://github.com/rob-murray/ui-message-queue.

Click the button to add some messages to the queue and watch them each displayed for a specific period of time.

**This file is pretty much the same as the demo but meant for local testing.
